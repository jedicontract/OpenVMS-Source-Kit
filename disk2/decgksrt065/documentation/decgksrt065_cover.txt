          Read Before Installing or Using DIGITAL GKS Version 6.5 for
          OpenVMS VAX Systems

          AV-PQPBJ-TE

          Whether you are a new or existing DIGITAL GKS[TM] (formerly
          DEC GKS[TM]) customer, please take time to read the following
          information about your product.

          Installation Information

          Installing DIGITAL GKS Version 6.5 requires the OpenVMS[TM]
          VAX[TM] Operating System Version 5.5-2 or higher, and DECwindows[TM] 
	  Motif[R] Version 1.1 or higher for OpenVMS VAX. Previous versions 
	  of DIGITAL GKS do not need to be installed before this version.

          Release Notes Information

          The release notes for DIGITAL GKS Version 6.5 contain installation-
          related instructions and a summary of technical changes, known
          problems, restrictions, and incompatibilities. To read the re-
          lease notes before installing DIGITAL GKS, invoke VMSINSTAL and
          follow the instructions in the installation guide. After DIGITAL
          GKS installation, the release notes are located in:

              SYS$HELP:DECGKS065.RELEASE_NOTES       (ASCII format)
              SYS$HELP:DECGKS065_RELEASE_NOTES.PS    (PostScript[R] format)

          License Management Facility

          Before you install DIGITAL GKS, you must register the Product
          Authorization Key (PAK), using the License Management Facility
          (LMF). See the OpenVMS License Management Utility Manual for
          registration instructions.

          Contents of This Kit

          o  Indented Bill Report (BIL) and Bill of Materials (BOM)

             Please read the BIL and BOM enclosed in this kit, and check
             to see that all items listed are actually in your kit. If
             your kit is damaged or any items are missing, call your
             DIGITAL representative.

          o  Media

             If you ordered media, you will find the media, this letter,
             and the Installing DIGITAL GKS for OpenVMS VAX Systems manual
             in this kit.

          o  DIGITAL GKS Documentation

             The documentation set for DIGITAL GKS includes the following
             manuals:

                DEC GKS User's Guide
                DEC GKS C Binding Reference Manual
                DEC GKS FORTRAN Binding Reference Manual
                DEC GKS GKS$ Binding Reference Manual
                DEC GKS GKS3D$ Binding Reference Manual
                Device Specifics Reference Manual for DEC GKS and DEC
                PHIGS

             The following manual is also available and can be ordered
             separately:

                Building a Device Handler System for DEC GKS and DEC
                PHIGS

             Optional documentation kits-which include the DEC GKS User's
             Guide, Device Specifics Reference Manual for DEC GKS and
             DEC PHIGS, and one binding reference manual-can also be
             purchased. To order these manuals, refer to the Software
             Product Description (SPD).

          o  Product Authorization Key (PAK) for OpenVMS

             The PAK contains information necessary to register DIGITAL
             GKS before you install it on an OpenVMS VAX system.

          o  Software Product Description (SPD)

             The SPD provides an overview of the DIGITAL GKS kit and its
             features, and describes the technical environment in which
             the product is supported. This document is provided on line.

          o  Software Performance Report (SPR)

             Use this form to report any problems with DIGITAL GKS, pro-
             vided you have purchased warranty services.

          Product Name Change

          With Version 6.2 of this product, DIGITAL changed the product
          name from DEC GKS to DIGITAL GKS.
