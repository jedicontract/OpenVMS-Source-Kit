Important Technical Information

Dear Customer:

The DECserver 300 Version 2.2C software is compatible with the DECserver 300 
hardware.  Version 2.2C of the DECserver 300 software incorporates the
following changes:
 
All patches issued against Version 2.2 of the DECserver 300 have been
incorporated into this release

For OpenVMS users, the installation kit includes several new DCL command files 
for DECserver management under DECnet/OSI

For additional information, refer to the DECserver 300 Software Release Notes.

If you are upgrading from Version 2.2 to Version 2.2C software, continue to 
use your existing software documentation set.  If you are a new customer
receiving Version 2.2C software, you will receive the following software
documents:

DECserver 300 Management

DECserver 300 Problem Solving

DECserver 300 Software Installation (MS-DOS, ULTRIX, or OpenVMS)

Terminal Server Commands and Messages
