          Features

          DIGITAL SNA Remote Job Entry for OpenVMS is a layered software
          product that allows OpenVMS[TM] systems to communicate with IBM®
          systems using the IBM Systems Network Architecture (SNA) Remote
          Job Entry (RJE) protocols.

          New features for Version 1.6 of DIGITAL SNA Remote Job Entry for
          OpenVMS include the following:

          o  OpenVMS V7.1 support

          o  All ECO changes since the release of Digital SNA Remote Job
             Entry for OpenVMS Version 1.5.

          For complete information about the product, refer to the DIGITAL
          SNA Remote Job Entry for OpenVMS Version 1.6 Software Product
          Description (SPD 26.85.07).

          Installation

          Register and load the product's PAK using the OpenVMS License
          Management Facility before you begin installation. Refer to
          the OpenVMS License Management Utility Manual, for complete
          licensing information.

          Before installing the product, please take a few minutes to
          read the DIGITAL SNA Remote Job Entry for OpenVMS software
          release notes. The Installation manual describes how to access
          the product's release notes using the VMSINSTAL utility.

          To install, enter the following:

          $ @SYS$UPDATE:VMSINSTAL SNARJE016 device-name OPTIONS N
