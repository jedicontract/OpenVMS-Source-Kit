DECnet[TM] SNA Gateway for Synchronous Transport

AV-JR51F-TE

Dear Customer:

This kit contains the DECnet[TM] SNA Gateway for Synchronous
Transport Version 1.3 software and the user documentation from
Version 1.2 (the documentation did not need to be revised for
Version 1.3). Please check the bill of materials to see that
all the items listed are in your distribution kit. If any item
is missing or damaged, please contact your Digital Equipment
Corporation representative.

DECnet SNA Gateway for Synchronous Transport Version 1.3 
supports OpenVMS[TM] Alpha and OpenVMS VAX[TM] Version 6.2 or
higher.

The DECnet SNA Gateway for Synchronous Transport documentation
set includes the following:

o  DECnet SNA Gateway for Synchronous Transport
   Installation (AA-JE89E-TE)

o  DECnet SNA Gateway for Synchronous Transport
   Guide to IBM[R] Parameters (AA-JE91E-TE)

o  DECnet SNA Gateway for Synchronous Transport
   Problem Solving (AA-MA17E-TE)

o  DECnet SNA Gateway for Channel Transport and Gateway for
   Synchronous Transport
   Management (AA-LU43E-TE)

In most cases, the DECnet SNA Gateway for Synchronous Transport
software operates the same on OpenVMS Alpha and OpenVMS
VAX systems. Any exceptions are noted in the release notes.
DIGITAL[TM] recommends that you read the release notes for any
technical issues before you begin installation. You can access
the release notes by responding to the prompts at the beginning
of the installation procedure, before you begin to install the
software.

To install the product, enter the following command:
$ @SYS$UPDATE:VMSINSTAL SNACST013 device-name OPTIONS N

If you need to order additional documentation, contact your 
DIGITAL sales representative or refer to the ordering information
in the Software Documentation Product Directory.
