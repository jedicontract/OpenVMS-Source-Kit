          Features

          DIGITAL SNA Application Programming Interface for OpenVMS is
          a layered software product that allows OpenVMS[TM] systems
          to communicate with IBM® systems using the IBM System Network
          Architecture (SNA) protocols.

          New features for Version 2.5 of DIGITAL SNA Application Program-
          ming Interface for OpenVMS include the following:

          o  OpenVMS V7.1 support

          o  All ECO changes since the release of Digital SNA Application
             Programming Interface for OpenVMS Version 2.4.

          For complete information about the product, refer to the DIGITAL
          SNA Application Programming Interface for OpenVMS Version 2.5
          Software Product Description (SPD 26.86.07).

          Installation

          Register and load the product's PAK using the OpenVMS License
          Management Facility before you begin installation. Refer to
          the OpenVMS License Management Utility Manual, for complete
          licensing information.

          Before installing the product, please take a few minutes to read
          the DIGITAL SNA Application Programming Interface for OpenVMS
          software release notes. The Installation manual describes how to
          access the product's release notes using the VMSINSTAL utility.

          To install, enter the following:

          $ @SYS$UPDATE:VMSINSTAL SNALU0025 device-name OPTIONS N
