          Dear Customer:

          Thank you for purchasing DIGITAL TCP/IP Services for OpenVMS
          (also known as UCX) networking software. This product is the
          DIGITAL implementation of TCP/IP protocols and services for the
          OpenVMS operating system.

          The product package contains the DIGITAL TCP/IP Services for
          OpenVMS Version 4.2 software and user documentation. Please
          check the Bill of Materials (BOM) to make sure you received all
          the items listed. If any item is missing or damaged, contact
          your DIGITAL representative.

          For a complete product description, see the DIGITAL TCP/IP
          Services for OpenVMS Version 4.2 Software Product Description
          (SPD 46.46.06).

          Features

          The DIGITAL TCP/IP Services for OpenVMS release notes outline
          the new features for Version 4.2. The release notes are located
          on the distribution media in the same directory as the software
          save sets. After installation, the release notes are located in
          SYS$HELP:UCX042.RELEASE_NOTES.

          Licenses

          The DIGITAL TCP/IP Services for OpenVMS product offers three
          licenses:

          o  Server license. Provides complete product functionality.

          o  Client license. Provides complete product functionality
             except for the following server components: NFS server,
             BIND server, PC-NFS server, remote BOOT server, and RMT/RCD
             server.

          o  Client upgrade license. Provides the server components for
             customers who purchased a client license and now require full
             server functionality.
