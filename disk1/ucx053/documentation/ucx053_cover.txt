          April 2002

          Dear Customer:

          Thank you for purchasing Compaq TCP/IP Services for OpenVMS
          networking software. The Compaq TCP/IP Services for OpenVMS
          software product is the OpenVMS implementation of the industry-
          standard TCP/IP protocol suite and internet services for OpenVMS
          Alpha and OpenVMS VAX systems.

          For a complete description of this product, see the Compaq
          TCP/IP Services for OpenVMS Version 5.3 Software Product De-
          scription (SPD 46.46.15).

          Features

          The Compaq TCP/IP Services for OpenVMS software is based on the
          Berkeley Software Distribution (BSD) Versions 4.3 and 4.4, with
          enhancements from Compaq Computer Corporation.

          This release of Compaq TCP/IP Services for OpenVMS features the
          following enhancements:

          o  Increased IPv6 functionality, including mobile correspondent
             node support and 4to6 transition

          o  Support for OpenVMS Extended File Specifications in FTP and
             NFS

          o  LPD clusterwide failover

          o  New IMAP mail server

          o  NTP Version 4

          o  BIND Version 9 server

          o  Kerberos authentication for TELNET connections

          o  UNIX subsystem configuration

          Release Notes

          The Compaq TCP/IP Services for OpenVMS Release Notes describes
          the problems fixed in this release. The release notes are lo-
          cated on the distribution media in the same directory as the
          software kits. After installation, the text version of the
          release notes is located in SYS$HELP:TCPIP053.RELEASE_NOTES.
          A PostScript version is also available in SYS$HELP:TCPIP053_
          RELEASE_NOTES.PS.

 






          Licenses

          The Compaq TCP/IP Services for OpenVMS product offers three
          licenses:

          o  Server license. Provides complete product functionality.

          o  Client license. Provides complete product functionality
             except for the following server components: NFS server, BIND
             server, PC-NFS server, remote BOOT server, RMT/RCD server,
             and IMAP server.

          o  Client upgrade license. Provides the server components for
             customers who purchased a client license and who now require
             full server functionality.

          Media Changes (VAX Only)

          This version of TCP/IP Services software for OpenVMS VAX (QA-
          VHRAA-H8) is available on CD-ROM only.
