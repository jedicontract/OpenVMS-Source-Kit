PRODUCT NAME:  BASEstar Classic DAS for Allen-Bradley INTERCHANGE Soft-
ware, Version 3.4A                                     SPD 64.68.01

DESCRIPTION

The BASEstar Classic DAS for Allen-Bradley INTERCHANGE Software al-
lows users to access Allen-Bradley Programmable Logic Controllers (PLCs)
through the BASEstar Classic Device Connect Manager (DCM).

Via BASEstar Classic DCM, the user may configure path, type, device
and physical point definitions, and also configure for polled and un-
solicited data collection.

The BASEstar Classic DAS for Allen-Bradley INTERCHANGE Software sup-
ports the following Allen-Bradley PLCs:

o  Allen-Bradley Pyramid Integrator Chassis

o  Allen-Bradley PLC-2 Family of devices (via DH from PI chassis)

o  Allen-Bradley PLC-3 Family of devices (via DH from PI chassis)

o  Allen-Bradley PLC-5 Family of devices (via DH+ from the PI chas-
   sis)

o  Allen-Bradley PLC-5/XXE devices

o  Allen-Bradley PLC-5/XXC devices (via 1785 ENET interface)

o  Allen-Bradley SLC-5/03 and SLC-5/04 devices (via DH+ from PI chas-
   sis)

o  Allen-Bradley SLC-5/05 devices

                              COMPAQ                    March 2000

 

The BASEstar Classic DAS for Allen-Bradley INTERCHANGE Software sup-
ports the following device access functions:

o  An Upload function that transfers the contents of device memory to
   a VMS file (Not supported by the PLC-5/250, PLC-2, PLC-3 or SLC de-
   vices)

o  A Download function that transfers the contents of a VMS file to
   device memory (Not supported by the PLC-5/250, PLC-2, PLC-3 or SLC
   devices)

o  A Simple Compare of device files (Not supported by the PLC-5/250,
   PLC-2, PLC-3 or SLC devices and not provided on OpenVMS/Alpha sys-
   tems)

o  A Read Data function that allows users and applications to read data
   from a specific address in device memory

o  A Write Data function that allows users and applications to write
   data to a specific address in device memory

o  A Read Status function that returns the following information:

   -  Processor type and communications interface model

   -  Operating mode and data table size for a PLC-2 system

   -  Operating mode, user program size, and memory size for a PLC-
      3 system

   -  Operating mode and memory size for a PLC-5 system

   -  Processor mode and processor size for a SLC system

o  Receive Unsolicited Data from a device
