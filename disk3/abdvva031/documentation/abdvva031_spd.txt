PRODUCT NAME:  BASEstar Open DAS for Allen-Bradley Data Highway,
 Version 3.1A                                              SPD 55.50.05

DESCRIPTION

The BASEstar Open Device Access Software (DAS) for Allen-Bradley Data
Highway allows users to access Allen-Bradley Programmable Logic Con-
trollers (PLCs) through BASEstar Open Device Services or BASEstar Open
Data Services.

Via BASEstar Open Device Services the user may configure device and
variable definitions, read and write variable values, and configure
for polled data collection and information reports.

BASEstar Open Data Services provide device independent access to shop
floor data allowing the user to configure data points, read and write
data point values and generate events based on data changes.

The BASEstar Open DAS for Allen-Bradley Data Highway supports the fol-
lowing Allen-Bradley PLCs:

o  Allen-Bradley PLC-2 Family

o  Allen-Bradley PLC-3 Family

o  Allen-Bradley PLC-5 Family

o  Allen-Bradley SLC Family

the DAS for Allen-Bradley:

o  BASEstar Open Server for OpenVMS VAX (development and runtime)

o  BASEstar Open Server for OpenVMS Alpha (development and runtime)

o  BASEstar Open Server for Tru64 UNIX (development and runtime)

Other Hardware Required:

Communication between the Allen-Bradley PLC and BASEstar computer re-
quire:

o  Allen-Bradley communications card, KE, KF, KF2, or KF3 (SLC pro-
   cessors only)

o  Compaq Serial input card or terminal server

o  Properly configured RS232-C cable
