PRODUCT NAME:  BASEstar Classic DAS for Siemens H1 Protocol, Version
3.4A                                                   SPD 61.41.01

DESCRIPTION

The BASEstar Classic DAS for Siemens H1 Protocol allows users to ac-
cess Siemens Programmable Controllers through the BASEstar Classic De-
vice Connect Manager (DCM).

Via BASEstar Classic DCM, the user may configure path, type, device
and physical point definitions, and configure for polled and unsolicited
data collection.

The BASEstar Classic DAS for Siemens H1 Protocol supports the follow-
ing Siemens Programmable Controllers:

o  S115U

o  S135U

o  S150U

o  S155U

The BASEstar Classic DAS for Siemens H1 Protocol supports the follow-
ing device access functions:

o  An Upload function that transfers the contents of a device's mem-
   ory to a VMS file (Not supported for the S155U device)

o  A Download function that transfers the contents of a VMS file to
   a device's memory (Not supported for the S155U device)

o  Start and Stop functions that change the operating mode of a pro-
   grammable controller (Not supported for the S155U device)

                              COMPAQ                    March 2000

 

o  A Read Data function that allows users and applications to read data
   from a specific address in device's memory

o  A write data function that allows users and applications to write
   data to a specific address in a device's memory

o  A read status function that returns the following information (Not
   supported for the S155U device):

   -  Status of the CP511 board

   -  Programmable controller type and software version

o  Receive Unsolicited Data from a device
