Dear BASEstar Customer,

Thank you for purchasing BASEstar Classic DAS for Allen-Bradley Data Highway 
V3.4A product.  This DAS allows BASEstar applications to connect to shop floor
devices in order to monitor and control manufacturing operations, and to 
automatically distribute manufacturing data collected from these devices 
throughout a distributed network of BASEstar systems.

For installation instructions, please refer to the Installation and User's 
Guide packaged with the documentation kits. The release notes also contain 
installation-related instructions, plus a summary of technical changes, known 
problems, restrictions, and incompatibilities.
