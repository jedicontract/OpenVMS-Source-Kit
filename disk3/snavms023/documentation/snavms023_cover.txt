Features

OpenVMS SNA is a layered software product that allows OpenVMS
VAX[TM] systems to communicate with IBM[R] systems using the IBM
System Network Architecture (SNA) protocols.

New features for Version 2.3 of OpenVMS SNA include the follow-
ing:

o  OpenVMS VAX V6.1 and V7.2 support

o  Support for remote DECnet[TM] client applications

Documentation

The OpenVMS SNA software documentation set includes the follow-
ing manuals:

o  OpenVMS SNA Installation

o  OpenVMS SNA Management

o  OpenVMS SNA Guide to IBM Parameters

o  OpenVMS SNA Problem Solving

Documentation is available in online format as part of the
OpenVMS VAX Online Documentation Library on CD-ROM (the Problem

you need to order any additional documentation, please contact
your DIGITAL sales representative or refer to the ordering
